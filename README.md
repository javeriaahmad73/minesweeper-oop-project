Minesweeper OOP Project 

This project is a Python implementation of the classic Minesweeper game, developed using Object-Oriented Programming (OOP) principles and the Tkinter library for the graphical user interface.

 Features
- Interactive GUI using Tkinter
- Random mine placement
- Left-click to reveal cells
- Right-click to mark potential mines
- Game over detection

 OOP Concepts Used
- Classes and Objects
- Constructors (`__init__`)
- Methods and Properties
- Encapsulation
- Interaction between multiple classes

Technologies Used
- Python
- Tkinter

 Project Structure
- `main.py` → Main game loop and UI layout
- `cell.py` → Cell logic and behavior
- `settings.py` → Game configuration
- `utilities.py` → Helper functions
